# Zeelool-style App (Expo + React Native + TypeScript)

Includes:
- Home screen (zeelool.com-style header + product grid)
- Product detail screen (carousel, tabs, specs, reviews)
- Try-On modal (like your screenshot) with photo upload using `expo-image-picker`

## Install & run
```bash
npm install
npm run ios
```

## If any missing Expo packages show up
```bash
npx expo doctor --fix-dependencies
```
